



# QurbaniHat – Livestock Booking Platform

QurbaniHat is a modern livestock marketplace web application where users can explore animals for Qurbani such as cows and goats, view detailed information, and place bookings after authentication.

---

## Live Website

https://a-08qurbani-hat.vercel.app/

---

## 🎯 Project Purpose

The purpose of QurbaniHat is to simplify the process of finding and booking livestock online during Qurbani/Eid-ul-Adha. Users can browse animals, check prices and details, and securely book animals through an authenticated platform.

---

##  Key Features

-  Authentication system using Better Auth
-  Email & Password Login/Register
-  Browse available livestock
- Animal details page
- Booking form for authenticated users
-  User profile update system
-  Fully responsive design
-  Featured animals slider using Swiper
-  Toast notifications for success/error handling
-  Google Sign In support
